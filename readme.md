## How to use
 - `programName --test` launch test
 - `programName --help` see commands

## Dependencies
- [googleTest](https://github.com/google/googletest)

## Install
Theses commands were only been tested on linux 
- sudo chmod 775 dependencies.sh
- sudo ./dependencies.sh;
- make all
- That's it !
